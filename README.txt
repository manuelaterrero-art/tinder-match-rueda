
TINDER MATCH RUEDA

Publicar en GitHub Pages:

1. Crear repositorio.
2. Subir index.html, styles.css y app.js.
3. Settings > Pages.
4. Deploy from Branch > main.
5. Guardar.

Para almacenamiento real:
- Crear Google Sheet.
- Crear Google Apps Script.
- Sustituir localStorage por fetch() al Apps Script.
